如果你想把租赁服中从 (1,2,3) 到 (4,5,6) 的区域导出
到名为 abc 的存档中，那么你可以使用下面的这个命令。

export/ (1,2,3) (4,5,6) abc

其中，abc 是存档的名字，它会生成在插件数据目录中下，
并且是一个文件夹，名字是 abc，然后里面放了存档的文件。

其他一些示例：
export/ (-59,-62,-264) (182,-36,65) 小城市
export/ (-1,-1,-1) (1,1,1) YTMKaP3lZwA=
export/ (-1,-65,-1) (-1,321,-1) YTMKaP3lZwA=

另外：
1. 存档文件夹是不能带有空格符号的，否则会出现问题
2. 要导出其他维度的建筑，请将机器人传送到对应维度
3. 不管机器人在哪个维度，建筑物总是生成在导出产物的主世界中

特别提醒：
1. 本插件按区块对齐，所以即便只指定了一个方块，也会导出这个方块对应的区块 (Y 轴不受影响)
2. 如果你导出了 (x, y, z) 附近的区域，你需要前往导出产物的 (x, y, z) 附近找到导出的建筑
